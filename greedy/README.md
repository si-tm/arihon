## 硬貨の問題
### 問題
1, 5, 10, 50, 100, 500円玉がそれぞれ、  
a, b, c, d, e, f枚あるとします。
できるだけ少ない枚数でA円支払いたいと考えています。
何枚の硬貨を出すことができるでしょうか？
なお、そのような支払い方は少なくとも1つは存在するとします。

### 制約
- 0 <= a, b, c, d, e <= pow(10, 9)
- 0 <= A <= pow(10, 9)

### 入力例
(a, b, c, d, e, f, A) = (3, 2, 1, 3, 0, 2, 620)
解答 : 6

## 区間スケジューリング問題
### 問題
n個の仕事があります。各仕事は
時間siに始まり、時間tiに終わります。
あなたは各仕事について、参加するかしないかを選ばなければなりません。
参加するならば初めから終わりまでいなければなりません。
また、参加する仕事の時間帯が重なってもいけません。
できるだけ多くの仕事に参加したいです。
何個の仕事に参加することができるでしょうか？

### 制約
- 1 <= N <= pow(10, 5)
- 1 <= si < ti <= pow(10, 9)

### 入力例
- n = 5  
- s = {1, 2, 4, 6, 8}
- t = {3, 5, 7, 9, 10}

## Best Cow Line
### 問題
N文字の文字列Sが与えられ、N文字の文字列Tを作ります。
初め、Tは長さ0の文字列で次のいずれかの操作が行えます。
- Sの先頭を1文字削除し、Tの末尾に追加する
- Sの末尾を1文字削除し、Tの末尾に追加する
辞書順比較でできるだけ小さくなるようにTを作ってください


### 制約
- 1 <= N <= 2000
- 文字列Sに含まれるのはローマ字の大文字のみ

### 入力例
- N = 6
- S = "ACDBCB"  
解答 : T = "ACDBCB"

### 解答  
先頭と末尾比較しまくる
