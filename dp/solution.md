# dp
## 01ナップサック問題
### **問題**
重さと価値がそれぞれwi, viであるようなn個の品物があります。
これらの品物からWを超えないように選んだときの価値の総和の最大値を求めなさい  
### **制約**  
- 1 <= n <= 100
- 1 <= wi, vi <= 100
- 1 <= W <= 10000  
### **解答**  

dp[i][j] := i個目までのうち重さがjを超えない最大となる価値  
遷移は、  

dp[i + 1][w] = 

- dp[i][w] (w < w[i])
- max(dp[i][w - w[i]] + v[i], dp[i][w]) (w >= w[i])

## 最長共通部分列問題
### **問題**
2つの文字列s1,s2...snとt1,t2...tmが与えられます。これら２つの共通部分文字列の長さの最大値を求めよ

### **制約**
- 1 <= n, m <= 1000
### **解答**

dp[i][j] := s[i]までとt[j]までの共通文字列の最大値  

遷移は、

dp[i][j] = max(dp[i][j - 1], dp[i - 1][j])
s[i] == t[j]の時、
dp[i][j] = max(dp[i][j - 1], dp[i - 1][j], dp[i - 1][j - 1] + 1)

dp[i][j] := i個目までのうち重さがjを超えない最大となる価値  
遷移は、  

dp[i + 1][w] = 

- dp[i][w] (w < w[i])
- max(dp[i + 1][w - w[i]] + v[i], dp[i][w]) (w >= w[i])
