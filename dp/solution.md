# dp
## 01ナップサック問題
### **問題**
重さと価値がそれぞれwi, viであるようなn個の品物があります。
これらの品物からWを超えないように選んだときの価値の総和の最大値を求めなさい  
### **制約**  
- 1 <= n <= 100
- 1 <= wi, vi <= 100
- 1 <= W <= 10000  
### **解答**  

dp[i][j] = i個目までのうち重さがjを超えない最大となる価値  
遷移は、  

dp[i + 1][w] = 

- dp[i][w] (w < w[i])
- max(dp[i][w - w[i]] + v[i], dp[i][w]) (w >= w[i])
